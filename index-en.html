How to find browser version number

Help for browser version number

Phone or Tablet
On a phone or tablet? No browser version required, just type in the name of the phone. For example: iPhone 6S or Samsung Galaxy.
Internet Explorer and Firefox
Internet Explorer & Firefox: use Help in menu bar, then About Internet Explorer or Alt-H-A (then Escape to exit)
Chrome
Chrome:type "chrome:help" into the address bar (then BACK to exit) or use the menu (icon is 3 horizontal lines, in browser address bar), then About Google Chrome.
Safari
Safari: use About Safari in the Safari menu.
